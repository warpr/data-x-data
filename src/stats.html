DATA×DATA

description

DATA×DATA is a minimal web analytics system.

Most web analytics systems are designed to track users, which may raise privacy concerns. DATA×DATA is designed to only log screen resolution and browser version for each page view, which is what a web developer needs to optimize their site.

No other data is logged. So no tracking cookies, ip addresses, geolocation data, etc... There is no way to associate a page view with a particular user†.

†Unless that user has a unique screen resolution or user agent string.
